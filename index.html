    Simpletag, a simple photo tagging plugin for Jquery

    Simple jquery photo tagging plugin. Based on imgareaselect plugin. Initial build, only based on what i need. Further development will include additional feature to simplify this whole module.


    Requirements :

    • Jquery (http://jquery.com/)
    • Imgareaselect (http://odyniec.net/projects/imgareaselect/)

    Optional(Included in download) :

    • Jquery Impromptu Plugin (http://trentrichardson.com/Impromptu/)
    • Javascript JSON (https://github.com/douglascrockford/JSON-js)

    Usage :

    1. Include required files
    2. Init simpletag 
      $("#image1").simpletag();
    3. Invoke add method to add tag to photo
