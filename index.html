    Simpletag, a simple photo tagging plugin for Jquery

    Simple jquery photo tagging plugin. Based on imgareaselect plugin. Initial build, only based on what i need. Further development will include additional feature to simplify this whole module.

    Requirements :

    • Jquery (http://jquery.com/)
    • Imgareaselect (http://odyniec.net/projects/imgareaselect/)

    Optional(Included in download) :

    • Jquery Impromptu Plugin (http://trentrichardson.com/Impromptu/)
    • Javascript JSON (https://github.com/douglascrockford/JSON-js)

    Usage :

    1. Include required files

      <link rel="stylesheet" type="text/css" href="css/imgareaselect-animated.css"/>
<script type="text/javascript" src="js/jquery-1.8.2.min.js"></script>
<script type="text/javascript" src="js/jquery.imgareaselect.pack.js"></script>
<script type="text/javascript" src="js/jquery.simpletag-0.2.js"></script>

    2. Init simpletag

      $("#image1").simpletag();

    3. Invoke add method to add tag to photo

      $("#image1").simpletag("add", "Label1");

    Documentation :

    Options

    • onSelectionComplete :
      (Optional) Callback function that will be trigger after done making selection. One argument will be supplied which is the selection coordinate. If impromptu is supplied, this will also return another one argument which is json data for form in the impromptu html. If this function return false, then selection will be cancel. Else, you need to return the label to be added to the selection.
    • initialZIndex :
      (Optional) Initial Z-Index for the first element. Next tag element will increment 1 value from this value.
    • existingList :
      (Optional) Existing json data of tag to be populate to the image. Format for data in json is as per below :
    var existingList = [{label:"Label 1", x1:100, y1:100, height:90, width:90}];
    • borderCss :
      (Optional) CSS for selection border EG: 2px #6E6E6E solid
    • borderRadius :
      (Optional) Radius for border. Default will be 5px if not specified.
    • autoHide :
      (Optional) Set this value to true will hide the selection after specified time. It will show again once you hover over the selection. Default will be false.
    • timeoutDelay :
      (Optional) Only use when autoHide is set to true. Delay before the selection dissapear.
    • impromptu :
      (Optional) This require additional 4 options which is :
      • enable :
        When this value set to true, simpletag will prompt for message upon making selection.
      • html :
        Html message to be shown in the prompt
      • selector :
        Selector for the element to get the HTML from. Can also include form in here.
      • callback :
        Callback function will be trigger after clicking ok button at the prompt. This is same like onSelectionComplete option. Except it will return two parameters which is the first one is the selection and second is the form object in JSON. You can refer example 3 in the demo page for the usage.

    Method :

    • add :
      Call this method to add new tag to the photo. When supplied argument to this method, the argument will be use as a label for the tag. Example :
    // This will create tag with Label 1 as a label.
$("#image1").simpletag("add", "Label 1");

// This can be use if impromptu is enable.
// Value for the tag element will be get from 
// impromptu.
$("#image1").simpletag("add");
